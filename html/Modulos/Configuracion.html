Mi primer proyecto

1. Para crear tu Primer Proyecto debes dirigirte al panel principal, buscar el módulo Configuración y hacer clic.

2. A continuación, seleccionaremos la opción Proyectos y Unidades.

3. Posteriormente, vamos a visualizar el botón Crear Proyecto, el cual debemos seleccionar.

4. Se abrirá una ventana donde vamos a ingresar los datos del proyecto: Nombre, Nombre de la Razón Social, CUIT, Dirección, Número de pisos, Nombre de la división x1 y x99, y fecha. También podremos agregar el logo del proyecto, este último punto es opcional.

5. Hacer clic en Guardar. ¡Listo! Tu proyecto ha sido creado y ya puedes comenzar a cargar las unidades.

Carga de las unidades de un proyecto

Una vez creado tu proyecto, vas a poder cargar las unidades, puedes realizarlo de manera individual o masivamente:

Carga Individual:

1. En la esquina superior derecha vas a encontrar el botón de Crear unidades, el cual vas a seleccionar:

2. Se abrirá una ventana para que ingreses todos los detalles de la unidad: Piso, N° de unidad, Tipo de unidad, precio en USD, Tipología, Superficie, etc.

*Campos obligatorios: Piso, N° de unidad, Tipo de unidad (verás algunas opciones por defecto, pero puedes crear más: Tipos de unidades).

Adicionalmente, podrás seleccionar si esa unidad está disponible para reventa y agregar todas las unidades que creas necesarias en el botón de Agregar Unidad.

3. Haces clic en Guardar y podrás ver la unidad creada.

Carga Masiva (Archivo Importador)

1. En la esquina superior derecha vas a encontrar el botón de Crear unidades, el cual vas a seleccionar:

2. Se abrirá una ventana para que ingreses el detalle de las unidades. En la parte inferior veremos un Botón Azul: Importar Unidades. Haremos clic en la flecha donde tendremos la opción de Descargar la Planilla Importadora.

3. En este archivo ingresaremos todos los detalles de nuestras unidades: Piso, N° de unidad, Tipo de unidad, precio en USD, Tipología, Superficie, Seleccionar si es para reventa o no, etc.

*Campos obligatorios: Piso, N° de unidad, Tipo de unidad (verás algunas opciones por defecto, pero puedes crear más: Tipos de unidades. Si agregas más unidades, debes volver a descargar la planilla).

4. Una vez completa la carga de las unidades, vas a volver a la plataforma para subir la planilla. Para ello, vas a hacer clic en el botón Importar Unidades, que te habilitará para seleccionar el archivo o arrastrarlo.

5. Una vez cargada la planilla, podrás ver todas las unidades ingresadas y haces clic en Guardar.

6. ¡Listo! Ya tienes cargadas en el sistema todas las unidades del proyecto.

Nota: Si quieres editar alguna unidad, solo debes hacer clic en los 3 puntitos y seleccionar la opción Ver detalle/Editar.

Configuración Rubros

La definición de rubros nos permitirá tener mejor orden e identificar de manera más precisa nuestros Proveedores y Gastos.

Losa0 traerá por defecto los rubros más comunes en el desarrollo inmobiliario, pero también vas a poder agregar o eliminar.

Nota: el sistema tiene una estructura de Árbol de Rubros y Subrubros.

1. Para configurar tus Rubros, debes dirigirte al panel principal, buscar el módulo Configuración y hacer clic.

2. Posteriormente, vas a ubicar la opción de Rubros, clic para ingresar.

3. Podrás visualizar el listado de Rubros.

4. Si quieres eliminar un rubro, busca los 3 puntitos, haz clic para desplegar las opciones y posteriormente clic en eliminar.

5. Si quieres agregar un nuevo rubro, en la parte superior derecha vas a ubicar el botón de Crear Rubro.

6. Se abrirá una ventana para ingresar todos los datos correspondientes: Nombre del Rubro, Tipo (Proveedor o Cliente).

Nota: Si el rubro a ingresar es un Rubro Padre, el campo Rubro padre se deja vacío. Si el Rubro a ingresar es un Subrubro, el campo Rubro padre debe contener el rubro al cual pertenecerá.

Importante: El sistema crea por defecto Cuentas contables de los rubros ubicadas en la cuenta contable Obra en curso. Si no quieren trabajar con este plan de cuentas, pueden crear la cuenta contable y agrupar los rubros en esta.

7. Una vez completada la información, clic en Guardar.

8. Listo, ya tienes configurados tus Rubros.

Configuración de bancos

La configuración de bancos es clave para que puedas tener todo el seguimiento de tus transacciones. Para esto, es necesario que se realice la configuración de bancos de manera manual o por medio de un archivo CSV.

1. Debes ingresar al panel principal, buscar la opción Configuración y hacer clic.

2. Ubicarás la opción Bancos, haz clic para ingresar.

3. Verás el listado de Bancos que tienes cargados en el sistema. Puedes eliminar el banco deseado, si lo quieres hacer, simplemente busca los 3 puntitos y elige eliminar.

4. Si deseas agregar un nuevo banco, puedes hacer clic en el botón Crear Banco.

5. Aparecerá una ventana donde podrás cargar los datos de: Nombre, Tipo de banco, CBU, CUIT y habilitar las cuentas que deseas cargar.

6. Una vez cargada toda la información, clic en Guardar.

7. Listo, ya tienes tus bancos configurados.
