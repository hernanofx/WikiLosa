Configurar el Mail para el Envío de Cuotas

El envío de cuotas es un módulo esencial que permite gestionar pagos de manera anticipada y automática para diferentes proyectos. Esto incluye la actualización de saldos y ajustes de índices según sea necesario.

La periodicidad del envío de cuotas se puede personalizar según las necesidades del proyecto, optimizando así la gestión financiera. Esto facilita mantener un control constante de los pagos.

Se puede incluir información relevante como detalles de cuenta corriente y CBU en los envíos, asegurando que los destinatarios tengan toda la información necesaria para los pagos. Esto mejora la comunicación con los usuarios.

El sistema permite elegir si se enviarán correos para distintas divisiones, adaptando así las notificaciones a las preferencias de cada proyecto. Esto brinda un mayor control sobre la comunicación.

Sincronizar la Carga de Leads vía Mail

Una vez configurado el envío de correos, se puede realizar un filtro exhaustivo para seleccionar proyectos y clientes específicos. Esto incluye opciones como enviar cuotas con saldo o en fechas determinadas.

Los usuarios pueden elegir enviar solo cuotas que tienen saldo, dejando afuera las que no tienen saldo. Esto ayuda a optimizar el proceso de envío.

El módulo permite seleccionar cuotas vencidas en períodos específicos, como las que vencen en noviembre. Así, se pueden enviar recordatorios precisos a los clientes.

Se puede personalizar el asunto del correo y elegir entre un informe simple o completo. Esto permite enviar información relevante según las necesidades del cliente.
